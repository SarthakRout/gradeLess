gradeLess

An Effortless Grading Solution

This project is a part of the course ESC101 : Fundamentals of Computing at Indian Institute of Technology , Kanpur.
gradeLess is an intelligent tool designed to evaluate hand-written answer copies using Machine Learning for Character Recognition effortlessly to save the precious time of instructors and tutors grading answers manually.
Currently, we support questions with binary answers (True / False) and Single/Multiple Choice Questions .
The Frontend is designed using CSS and JS (for the beauty part) and Angular 8 for most of the actual work ,that is evaluation of answer copies.
The Backend is developed using Express.js and it is middleware between our Frontend and Python program which uses OpenCV for image processing and employs Tesseract Engine which uses Machine Learning Algorithms for correctly evaluating the answers.
